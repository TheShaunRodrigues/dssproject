DSS for Hate Speech Detection

About this site

This is a website that we have created to simulate the workings of a social media platform, that only takes in the values for username and the entered text data, and upon clicking the submit button, stores the values either in a database or in a local variable for further comparisons.
If offensive or hateful speech is detected by the program, it assigns a strike modifier to the registered username, or increments the value if it already exists. If the strikes against the username hit three, the user who it is against gets restricted and is not allowed to post any further text on the site
